# scan_kartu_ujian_digital
Aplikasi Scan Kartu Ujian memanfaatkan html5-qrcode untuk memindai QR secara real-time. Setelah scan berhasil, data dikonfirmasi lewat SweetAlert lalu dikirim ke server via AJAX. Sistem juga mengirim notifikasi otomatis ke peserta melalui WhatsApp API tanpa memuat ulang halaman.
